# install-mcp [(view live)](https://adamjones.me/install-mcp/)

Shows install instructions for MCP servers across different clients. Pass a server URL via query params and pick your client from a dropdown.

## Contributing

Pull requests are welcomed on GitHub! To get started:

1. Install Git and Node.js
2. Clone the repository
3. Install dependencies with `npm install`
4. Run the app with `npm start`
5. Build with `npm run build`

## Releases

Commits to the master branch are automatically published to GitHub Pages.
